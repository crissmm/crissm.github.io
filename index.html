Type&Strike

Type&Strike is a 2D arcade endless game for Android and iOS developed with Unity(C#). The aim is to destroy the enemy spaceships by typing with your keyboard the word they carry. So the player has to type with his/her keyboard as fast as possible to protect himself/herself of the enemies and smash them! The player can share with his/her Facebook friends the score reached and try to be the best!



Run4Fun

Run4Fun is a serious game for PC and Android developed with Unity3D(C#) and listed as a platform runner video game. The serious goal is to make aware the player of the risks associated with drug use. The project received Honours by Universitat Ramon Llull.

Gaze: A seamless way to interact with objects

Ideation, development and validation of a new interaction mode, which main goal is to improve the relationship between everyday objects and people, from a natural interaction that allows a person to use the functionality of the objects using his/her gaze. The project was developed with Google Glass (Java) and received Honours by Universitat Ramon Llull.

BCN Sounds

Project exposed in the MWCentre in Barcelona for 4 months. By touching different parts of this map of Barcelona with the hand, the user is able to discover the sounds of different places around the city. This is a poster printed using conductive ink, which allows electric circuits to be designed on paper. An electrical current of millions of electrons passes through the ink. So we converted the paper into a capacitive sensor to detect when the user's hand touches the ink. You can check out the project in the MWCentre webpage.

  • Client: MWCentre - Universitat Ramon Llull.
